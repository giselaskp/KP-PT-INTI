# Asset Management System

Laravel-based Asset Management System developed during Internship at PT INTI (Persero).

---

## Features

### Authentication
- Login
- Register
- Logout

### Asset Management
- Add Asset
- Edit Asset
- Delete Asset
- Search Asset
- Pagination
- Asset Image Upload
- Image Preview
- Drag & Drop Image Upload
- Image Validation (JPG, JPEG, PNG • Max 5 MB)

### Dashboard
- Asset Statistics
- Responsive Sidebar
- Asset Table
- Image Preview Modal
