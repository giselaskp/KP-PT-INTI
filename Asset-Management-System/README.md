# Week 1 - CRUD Laravel

## Objective

Develop a web-based Asset Management System using Laravel and MySQL to manage company asset data through CRUD operations.

---

## Features

### Authentication
- Login
- Register
- Logout

### Asset Management
- Add Asset
- Edit Asset
- Delete Asset
- Search Asset
- Pagination
- Asset Image Upload
- Image Preview
- Drag & Drop Image Upload
- Image Validation (JPG, JPEG, PNG • Max 5 MB)

### Dashboard
- Asset Statistics
- Responsive Sidebar
- Asset Table
- Image Preview Modal

---

## Tech Stack

- Laravel 12
- PHP 8
- Bootstrap 5
- MySQL
- Blade Template
- JavaScript
- Bootstrap Icons
